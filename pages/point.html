THQN Algorithm Explorer

Two Pointers (2 con trỏ) là một kỹ thuật được sử dụng phổ biến trong lập trình thi đấu để giải quyết các bài toán tối ưu hóa, tìm kiếm và sắp xếp mảng. Ý tưởng của kỹ thuật này là sử dụng hai con trỏ chạy trên mảng với các quy tắc cụ thể để giải quyết bài toán.



Kỹ thuật Two Pointers được sử dụng trong nhiều bài toán như tìm kiếm cặp số có tổng bằng một số x, tìm kiếm ba số có tổng bằng một số x, tìm kiếm đoạn con có tổng lớn nhất, tìm kiếm đoạn con có tổng bằng một số x, v.v. Kỹ thuật này cũng có thể được kết hợp với các phương pháp khác để giải quyết các bài toán phức tạp hơn.



  • Hai con trỏ thường được đặt ở hai đầu mảng hoặc ở các vị trí khác nhau trên mảng.


  • Các con trỏ cùng di chuyển hoặc di chuyển theo quy tắc cụ thể dựa trên yêu cầu của bài toán.


  • Tại mỗi bước, ta kiểm tra điều kiện để di chuyển con trỏ sao cho đạt được kết quả mong muốn.


  • Kỹ thuật Two Pointers thường giúp giảm đáng kể độ phức tạp thời gian so với các phương pháp tìm kiếm và sắp xếp thông thường.


  • Example:


    • #include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

vector<int> twoSum(vector<int>& nums, int target) {
    vector<int> result;
    int left = 0;
    int right = nums.size() - 1;

    // Sắp xếp mảng
    sort(nums.begin(), nums.end());

    while (left < right) {
        int sum = nums[left] + nums[right];
        if (sum == target) {
            // Tìm thấy kết quả, lưu vào vector result
            result.push_back(nums[left]);
            result.push_back(nums[right]);
            return result;
        } else if (sum < target) {
            // Tăng con trỏ left nếu tổng nhỏ hơn target
            left++;
        } else {
            // Giảm con trỏ right nếu tổng lớn hơn target
            right--;
        }
    }

    // Không tìm thấy kết quả
    return result;
}

int main() {
    vector<int> nums = {2, 7, 11, 15};
    int target = 9;

    vector<int> result = twoSum(nums, target);

    if (result.size() == 2) {
        cout << "Hai so co tong bang " << target << " la: " << result[0] << " va " << result[1];
    } else {
        cout << "Khong tim thay hai so co tong bang " << target << " trong mang.";
    }

    return 0;
}


  • Độ phức tạp thời gian: \(O(n\log n)\) cho việc sắp xếp mảng và \(O(n)\) cho kỹ thuật 2 con trỏ


  • Độ phức tạp không gian: \(O(1)\). Thuật toán chỉ sử dụng một số biến cố định không phụ thuộc vào kích thước đầu vào. Không có bất kỳ cấu trúc dữ liệu bổ sung nào được sử dụng trong thuật toán.
