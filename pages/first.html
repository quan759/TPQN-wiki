Dưới đây là ví dụ về chương trình C++ đơn giản nhất: 



#include <iostream>
using namespace std;
int main() {
    cout << "Hello, World!" << endl;
    return 0;
}


Trong chương trình trên, chúng ta sử dụng câu lệnh #include <iostream> để bao gồm thư viện iostream của C++. Thư viện này cung cấp các công cụ cho việc nhập và xuất dữ liệu.



Dòng int main() là điểm bắt đầu của chương trình. Tất cả mã C++ chạy trong hàm main(). 'using namespace std;' Dùng để gọi trực tiếp các thành phần của không gian tên "std", tức là thay vì phải viết std::cout thì chúng ta chỉ cần viết cout.



Trong ví dụ này, chúng ta sử dụng cout để xuất chuỗi "Hello, World!" ra màn hình và endl để thêm dấu xuống dòng.



Cuối cùng, chúng ta sử dụng return 0; để kết thúc chương trình và trả về giá trị 0. Return 0 còn có công dụng kết thúc chương trình.



  • Input and Output:


    • Trong lập trình thi đấu, chúng ta thường sử dụng cin để nhập dữ liệu từ bàn phím và cout để xuất dữ liệu ra màn hình. Dưới đây là ví dụ: 



    #include <iostream>
using namespace std;
int main() {
    int x;
    cout << "Enter a number: ";
    cin >> x;
    cout << "You entered: " << x << std::endl;
    return 0;
}


    Trong ví dụ trên, chúng ta khai báo biến x để lưu trữ số nguyên nhập từ bàn phím. Sau đó, chúng ta sử dụng cin để nhập giá trị vào biến x. Cuối cùng, chúng ta sử dụng cout để xuất giá trị của biến x ra màn hình.



  • Variables and Data Types:


    • Trong C++, chúng ta phải khai báo biến trước khi sử dụng. Mỗi biến có một kiểu dữ liệu xác định loại dữ liệu mà nó có thể chứa. Dưới đây là ví dụ về khai báo biến và các kiểu dữ liệu cơ bản: 



    #include <iostream>
using namespace std;
int main() {
    int age = 25; // Kiểu dữ liệu số nguyên
    double height = 1.75; // Kiểu dữ liệu số thực
    char grade = 'A'; // Kiểu dữ liệu ký tự
    bool isStudent = true; // Kiểu dữ liệu logic (đúng/sai)
    
    cout << "Age: " << age << endl;
    cout << "Height: " << height << endl;
    cout << "Grade: " << grade << endl;
    cout << "Is student: " << isStudent << endl;
    
    return 0;
}


    Trong ví dụ trên, chúng ta khai báo các biến age, height, grade và isStudent với các kiểu dữ liệu tương ứng. Sau đó, chúng ta sử dụng cout để xuất giá trị của các biến ra màn hình.



  • Operators:


    • Trong C++, chúng ta có nhiều loại toán tử để thực hiện các phép tính. Dưới đây là ví dụ về sử dụng các toán tử cơ bản: 



    #include <iostream>
using namespace std;
int main() {
    int a = 5, b = 3;
    
    int sum = a + b; // Phép cộng
    int diff = a - b; // Phép trừ
    int product = a * b; // Phép nhân
    int quotient = a / b; // Phép chia (chia lấy phần nguyên)
    int remainder = a % b; // Phép chia (lấy phần dư)
    
    cout << "Sum: " << sum << endl;
    cout << "Difference: " << diff << endl;
    cout << "Product: " << product << endl;
    cout << "Quotient: " << quotient << endl;
    cout << "Remainder: " << remainder << endl;
    
    return 0;
}


    Trong ví dụ trên, chúng ta khai báo hai biến a và b. Sau đó, chúng ta sử dụng các toán tử +, -, *, / và % để thực hiện các phép tính và gán kết quả vào các biến tương ứng. Cuối cùng, chúng ta sử dụng cout để xuất giá trị của các biến ra màn hình.



  • Selection Statements


    • Cấu trúc rẽ nhánh if-else là một cấu trúc điều khiển được sử dụng trong lập trình để kiểm tra điều kiện và thực thi các khối mã khác nhau dựa trên kết quả của điều kiện. Nó cho phép bạn thực hiện các hành động khác nhau tùy thuộc vào điều kiện được xác định. 



    #include <iostream>
using namespace std;
int main() {
    int a;
    cin >> a;
    if (a == 1) {
       cout << 1 << endl;   
    }
    else if (a == 2) {
       cout << 2 << endl;
    }
    else {
       cout << 3 << endl;
    }
    
    return 0;
}


    Trong ví dụ trên, chúng ta khai báo biến a và yêu cầu người dùng nhập giá trị cho biến a. Nếu người dùng nhập 1, màn hình sẽ in ra số 1, nếu người dùng nhập 2, màn hình sẽ in ra số 2, nếu người dùng nhập 1 số khác 1 và 2, màn hình sẽ in ra số 3.



  • Operators in Selection Statements:


    • Trong cấu trúc rẽ nhánh if else, có nhiều toán tử dùng để so sánh và kiểm tra điều kiện đúng sai:



  • == : dùng để kiểm tra xem 2 giá trị có bằng nhau hay không


  • <, >: dùng để kiểm tra xem giá trị thứ nhất nhỏ hơn hay lớn hơn giá trị thứ 2


  • <=, >=: dùng để kiểm tra xem giá trị thứ nhất có lớn hơn hoặc bằng, nhỏ hơn hoặc bằng giá trị thứ 2 hay không

using namespace std;
